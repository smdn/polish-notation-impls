# How to run tests
## Prerequisites
- .NET SDK 10 or over (required by [tools/format-xml.cs](tools/format-xml.cs))
- PowerShell (required by [run-test.ps1](run-test.ps1))
- node (required to run [demo](../../src/demo/contents/))
- npm (required by [generate-svg.js](generate-svg.js), see also [package.json](package.json))

## Install dependencies
```sh
npm install
```

## Run test script
```sh
./run-test.ps1
```
